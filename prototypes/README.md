# Prototypes (all archived)

**The game now lives in `app/index.html`** — these files are development history, kept for reference, not maintained.

- `take-47.html` — the scene-progression prototype that became the app (superseded 2026-07-03)
- `stick-the-landing.html` — v1 concept prototype (bake-off winner)
- `stick-the-landing-v2.html` — v2 with ragdoll/juice/sound (pre-theme)
- `squeeze.html` — the other bake-off concept, parked as a backup idea (see BACKLOG.md)

All are single-file HTML5 canvas games: double-click to play in a browser. Space bar = tap.
